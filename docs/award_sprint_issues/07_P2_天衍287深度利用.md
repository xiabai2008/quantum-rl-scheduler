## 任务描述

充分利用比赛提供的"祖冲之三号"同款287比特超导量子计算机真机算力，开展深度实验。真机数据是核心差异化优势——大部分队伍可能只用仿真，而我们有287比特真机数据。

**为什么是亮点**：
- 比赛免费提供天衍-287真机算力，不用白不用
- 287比特真机实验数据极具说服力
- 真机能体现方案的工程落地能力
- 是"技术创新性"和"验证严谨性"维度的加分项

## 工作内容

### 1. 真机要时申请与规划
- [ ] 申请比赛提供的真机要时（通过赛事邮箱 tianyan@chinatelecom.cn）
- [ ] 制定真机实验计划，高效利用机时
- [ ] 与赛事保障组沟通，了解排队情况和最佳使用时段

### 2. 天衍-287 大规模任务调度实验
- [ ] **完整RL调度实验**：在287比特真机上运行完整的RL调度流程
  - 任务规模：32/64/128个量子任务
  - 调度策略：PPO vs FCFS vs SJF
  - 对比指标：总执行时间、资源利用率、等待时间
- [ ] **仿真 vs 真机差异分析**：
  - 相同任务在仿真和真机上的性能对比
  - 噪声对调度效果的影响
  - Mock校准偏差验证

### 3. 噪声自适应调度验证
- [ ] 验证系统在真实噪声环境下的调度性能优势
- [ ] 不同噪声水平下的调度策略适应性
- [ ] 噪声感知调度 vs 无噪声感知调度对比

### 4. 多比特任务调度
- [ ] 从1-qubit扩展到多qubit任务（2-qubit, 4-qubit, 8-qubit）
- [ ] 展示系统对真实量子任务的适配能力
- [ ] 不同比特数任务的混合调度性能

### 5. 真机时间线记录

## 真机额度预估
- 小规模测试：~50个任务 × 10分钟 = ~8小时
- 正式实验：~200个任务 × 15分钟 = ~50小时
- 总计：约 60 小时真机能时
- **建议**：分阶段申请，先申请20小时做测试
