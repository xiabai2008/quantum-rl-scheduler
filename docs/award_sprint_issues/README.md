## 🚀 批量创建 GitHub Issues

### 方法一：使用脚本（推荐）

```bash
# 1. 设置 GitHub Token
# Windows PowerShell:
$env:GITHUB_TOKEN="your_personal_access_token"

# Linux/Mac:
export GITHUB_TOKEN="your_personal_access_token"

# 2. 预览（不实际创建）
python scripts/create_award_issues.py --dry-run

# 3. 实际创建
python scripts/create_award_issues.py
```

### 方法二：手动创建

直接在 GitHub Issues 页面手动创建，复制对应 markdown 文件的内容即可。

### 获取 GitHub Token

1. 访问 https://github.com/settings/tokens
2. 生成 Fine-grained personal access token
3. 权限选择：Issues → Read and write
4. 复制 token 备用
