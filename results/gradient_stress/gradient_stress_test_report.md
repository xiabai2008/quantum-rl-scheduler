1. **FCFS/SJF 在所有规模下表现稳健**，两者差距 < 2%，是可靠的传统基线
2. **Quantum-Only 和 Classical-Only 随规模增大灾难性退化**：

   - 1000 任务: QO=-29753.6, CO=-30035.7（完全不可用）

3. **Greedy 策略在所有规模下均劣于 FCFS**，且随规模退化显著（100→1000：235→-9490）
4. **Random 策略在 1000 任务时仅获 2641.5 奖励**，远低于 FCFS 的 6669.8（-60.4%），差距随规模扩大

### 核心洞察

- 在 **100-1000 任务梯度上**，FCFS 作为基线表现出良好的可扩展性
- Quantum-Only/Classical-Only 在超过 200 任务后急速恶化，验证了混合调度（量子+经典）的必要性
